# AI Advanced Colab Course

A static landing page for Archbishop Molloy High School senior AI course notebooks.

## Contents

- `index.html` — course homepage
- `styles.css` — visual styling for the site

## Deployment

This project can be hosted as a static site on Netlify.

### To publish

1. Push this repository to GitHub.
2. Connect the repository to Netlify.
3. Set the build command to `n/a` and publish directory to `/`.

## Notes

Replace each chapter card `href` in `index.html` with the actual Google Colab notebook URL once available.
